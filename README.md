# DEPI-Project

## Milestone 4: MLOps, Monitoring & Web Dashboard

### Objectives

• Implement MLOps best practices for lifecycle management and business integration

### Tasks

1. **Experiment Tracking**
   a. Use MLflow or Azure ML tracking for models and metrics
2. **Web Dashboard**
   a. Create a dashboard (Power BI or Streamlit) showing churn predictions & business KPIs
3. **Model Monitoring**
   a. Set drift detection triggers, performance alerts
4. **Automated Retraining Strategy**
   a. Schedule pipelines to retrain the model on fresh data based on drift metrics

### Deliverables

• Monitoring & Retraining Pipeline
• MLOps Documentation
• Interactive Dashboard
